Обращайся к селекторам

Базовые селекторы: коллекции

  • $('.class') - по классу
  • $('#identifer') - по id
  • $('element') - по тегу
  • $('button, .btn, #request') - по набору селекторов

Обход DOM-дерева

  • .find('.class') - потомки
  • .filter('.class') - уточнение поиска
  • .children() - дети
  • .parent() - родитель
  • .closest() - предок
  • .siblings() - соседи
  • .next() - следующий
  • .prev() - предыдущий

Фильтрация селекторов

  • .eq(3) и $(':eq(3)')- второй элемент
  • $(':even') - нечётные элементы
  • $(':odd') - чётные элементы
  • $(':first') и .first() - первый элемент
  • $(':last') и .last() - последний элемент
  • $(':not(.class)') - исключающий селектор
  • $('button, .btn, #request') - по набору селекторов

Прячь и показывай

  • Базовые: .show(), .hide(), .toggle()
    функции отображения/скрытия
  • Прозрачность: .fadeIn(), .fadeOut(), .fadeToggle()
    плавное отображения/скрытие через изменение прозрачности
  • Слайдинг: .slideDown(), .slideUp(), .slideToggle()
    плавное отображения/скрытие через изменение высоты

Пример: $('.class').hide()

Манипулируй

  • .html('<h1>хм</h1>') - замена html внутри текущего элемента
  • .text('текст') - изменение содержимого с экранированием
  • .remove() - удаляем текущий html-элемент
  • .empty() - удаляем содержимое текущего html-элемента

Помещай в элемент

  • .append('<h1>хм</h1>') - вставка html в конец текущего
  • $('<h1>хм</h1>').appendTo('.class') - вставка html в конец указанного
  • .prepend('<h1>хм</h1>') - вставка html в начало текущего
  • $('<h1>хм</h1>').prependTo('.class') - вставка html в начало указанного

Стилизуй на лету

  • .css('color') - получаем css-свойство
  • .css('color', 'red') - применяем css-свойство
    по сути аналог .attr('style', 'color: red')
  • .css({'color': 'red', 'background-color': 'blue'}) - массовое изменение значений css-свойств
  • .width() и .height() - полученаем размеры

Изменяй классы

  • .addClass('btn') - добавляем класс btn
  • .hasClass('btn') - проверяем наличие класса btn
  • .removeClass('btn') - удаляем класс btn
  • .toggleClass('btn') - добавляем класс btn, если его нет. удаляем, если есть

            function blinkme(event){

              event.preventDefault();

            }

            $(".button").on("click", blinkme);
